Política de Privacidade - Caixinha de Promessas

Este aplicativo respeita sua privacidade. Nenhum dado pessoal é coletado, armazenado ou compartilhado com terceiros.

Dados coletados

O aplicativo Caixinha de Promessas não coleta dados sensíveis, de localização, nem informações pessoais dos usuários. Os versículos acessados ficam salvos apenas no próprio aparelho do usuário para uso offline.

Compartilhamento de dados

Não compartilhamos nenhuma informação com terceiros.

Segurança

Nosso app não armazena dados na nuvem. Todo conteúdo é local, oferecendo maior privacidade.
